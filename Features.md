# Features

## Summary

The components of the input $X$. Usually $X$ is multi-dimensional. The tumor example extends to two features (tumor size + patient age → a 2D plot with a boundary separating the classes). Real breast-cancer datasets use many features (clump thickness, uniformity of cell size, uniformity of cell shape, adhesion) - too high-dimensional to plot. Later, SVMs + kernels let you represent a point with an infinite-dimensional feature vector without storing it.
