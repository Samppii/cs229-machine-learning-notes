# Well-posed learning problem

## Summary

**Tom Mitchell's definition:** a program *learns* from experience $E$ with respect to task $T$ and performance measure $P$ if its performance on $T$, as measured by $P$, improves with experience $E$.

Checkers example:
- $E$ = games played against itself (self-play)
- $T$ = playing checkers
- $P$ = win rate against the next opponent
