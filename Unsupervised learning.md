# Unsupervised learning

## Summary

You are given only $X$ - **no labels** - and must find interesting structure in the data.

- [[Clustering]] (K-means)
- [[Independent components analysis]] (the cocktail party problem)

Word analogies also emerge from unlabeled internet text (man:woman :: king:queen; Tokyo:Japan :: DC:USA).

## Notes

- Most recent *economic* value has come from [[Supervised learning]], but unsupervised learning has important uses and remains a research frontier.
